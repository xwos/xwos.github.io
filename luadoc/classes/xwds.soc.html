xwds.soc:pin (soc, pin)

Returns:

Usage:

xwds.soc:req (soc, port, pinmsk)
申请GPIO

注意：序号是从0开始编号。

Parameters:

  • soc userdata (in) C代码中注册的SOC

  • port number GPIO PORT的序号

  • pinmsk number GPIO PIN的掩码

Returns:

    number
    • ● rc 此方法的返回值
      • ○ 0 没有错误
      • ○ -ERANGE GPIO PORT错误
      • ○ -EBUSY GPIO PIN已被使用

Usage:

    rc = stm32:req(4, stm32:pin(7)) -- GPIOE PIN7

同时输出多个GPIO PIN

  • port number GPIO PORT的序号

  • pinmsk number GPIO PIN的掩码

  • out number GPIO 输出值

Returns:

    number
    • ● rc 此方法的返回值
      • ○ 0 没有错误

      • ○ -EPERM GPIO PIN未被使用

xwds.soc:input (soc, port, pinmsk)
同时从GPIO读取多个PIN的输入值

注意：序号是从0开始编号。

Parameters:

  • soc userdata (in) C代码中注册的SOC

  • port number GPIO PORT的序号

  • pinmsk number GPIO PIN的掩码

Returns:

    {number,number}
    • ● rc 此方法的返回值
      • ○ 0 没有错误
      • ○ -ERANGE GPIO PORT错误
      • ○ -EPERM GPIO PIN未被使用
    • ● in 输入值

Usage:

    -- 读取GPIOC，掩码0xFFFFFFFF
rc, in = stm32:input(stm32, 2, 0xFFFFFFFF)
