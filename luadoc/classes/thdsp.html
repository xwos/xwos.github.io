Class thdsp

XWLUA类：线程对象强指针

线程对象强指针是Lua语言中的一种 userdata ，用于 强引用 XWOS的线程对象。

强引用 表示会增加XWOS的线程对象的 引用计数 。

中断线程的阻塞态和睡眠态

thdsp:quit (thdsp)
通知线程退出

此方法用于线程 A 向另一个线程 B 设置 退出 状态。
　● 线程 B 是否要退出，什么时候退出，由线程 B 自己决定；
　● 调用此方法的线程 A 也不会等待子线程退出；
　● 线程 B 可以是 Joinable 的，也可以是 Detached 的；
　● 此方法可被重复调用，线程 B 的 退出 状态一旦被设置，不可被清除。

Parameters:

  • thdsp userdata (in) 线程对象强指针

Returns:

    number 错误码
    　● 0 没有错误

Usage:

    rc = thdsp:quit()
thdsp:join (thdsp)
等待线程结束，回收线程内存资源，抛出它的返回值

此方法类似于POSIX线程库的 pthread_join() ，用于等待另一个线程退出。 线程 A 调用此方法时，会阻塞等待线程 B 退出。线程 B 退出后，会释放线程 B 的内存资源，然后抛出线程 B 的返回值。

此方法只能对 Joinable 的线程 B 使用。

Parameters:

  • thdsp userdata (in) 线程对象强指针

Returns:

  1. number 错误码
    　● 0 没有错误
    　● -EINVAL 线程不是 Joinable 的
  2. number 子线程抛出的返回值

Usage:

    rc, childrc = thdsp:join()
thdsp:stop (thdsp)
终止线程并等待它退出，回收线程内存资源，并抛出线程的返回值

此方法用于线程 A 停止另一个线程 B ，并等待线程 B 结束。
　● 此方法只能对 Joinable 的线程 B 使用。
　● 线程 A 调用此方法时，会对线程 B 设置一个 退出 状态，然后一直阻塞等待线程 B 退出。
　● 线程 B 是否要退出，什么时候退出，由线程 B 自己决定。
　● 当线程 B 退出后，此方法会释放线程 B 的内存资源，然后抛出线程 B 的返回值。
　● 此方法等价于 thdsp:quit() + thdsp:join() 的组合。

Parameters:

  • thdsp userdata (in) 线程对象强指针

Returns:

  1. number 错误码
    　● 0 没有错误
    　● -EINVAL 线程不是Joinable的
  2. number 子线程抛出的返回值

Usage:

    rc, childrc = thdsp:stop()

Parameters:

  • thdsp userdata (in) 线程对象强指针

Returns:

    number 错误码
    　● 0 没有错误

Usage:

    rc = thdsp:detach()
