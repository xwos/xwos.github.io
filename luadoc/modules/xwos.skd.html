Functions

id ()
获取当前CPU的ID

Returns:

    number CPU的ID

Usage:

    id = xwos.skd.id()
tt ()
获取当前CPU调度器的系统滴答时间

XWOS的系统滴答时间是一个有符号的64位整数，以纳秒为单位。

Returns:

    number 系统滴答时间

Usage:

    time = xwos.skd.tt()
tc ()
获取当前CPU调度器的系统滴答计数

XWOS的系统滴答计数表示滴答定时器中断了多少次。

Returns:

    number 系统滴答计数

Usage:

    count = xwos.skd.tc()
ts ()
获取当前CPU调度器的系统时间戳

XWOS的系统时间戳 = XWOS的系统滴答时间 + 滴答定时器中数值换算成纳秒的时间。

Returns:

    number 系统时间戳

Usage:

    timestamp = xwos.skd.ts()
