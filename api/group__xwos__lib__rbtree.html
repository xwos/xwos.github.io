详细描述

红黑树性质

  • 1. 节点不是红色就是黑色；
  • 2. 根节点是黑色；
  • 3. 所有的叶子都是黑色；
  • 4. 每个红色的节点，如果有子节点，必须是黑色；
  • 5. 从任意节点到其任意子树的叶子的路径上，所包含的黑色节点数量相同。

算法特点

  • 红黑树节点的内存地址必须按4字节对齐，其地址信息的低两位必须为0， 算法使用这两个空闲的位存放位置(pos)和颜色(color)信息。
