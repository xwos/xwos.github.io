XWOS RUST：内存管理

XWOS已经在C语言层面已经实现了libc的malloc()、free()、memalign()、realloc()等一系列函数， XWOS RUST的内存分配器只是对这些函数进行封装。

用户需要在xwrust_main()函数之前设置全局内存分配器，例如：

#[global_allocator]
pub static ALLOCATOR: xwrust::xwmm::Allocator = xwrust::xwmm::Allocator;

#[no_mangle]
pub unsafe extern "C" fn xwrust_main() {
    // 主线程代码
}
