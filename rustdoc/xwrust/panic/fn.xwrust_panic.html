XWOS RUST的panic处理函数

此函数会调用board_panic()交由用户处理， 用户需要在APP模块中实现fn board_panic(info: &PanicInfo) -> !;。

panic处理函数常用于打印程序的backtrace。

在MCU开发中，调试器会帮我们完成这一点。因此可以通过一条断点指令实现此函数。 例如基于ARM-M的MCU可以这样实现：

use cortex_m::asm;
use core::panic::PanicInfo;

pub fn board_panic(_info: &PanicInfo) -> ! {
    loop {
        asm::bkpt();
    }
}
