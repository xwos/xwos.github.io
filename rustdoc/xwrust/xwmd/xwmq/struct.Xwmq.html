source

pub fn init<'a>(&'a self) -> (XwmqTx<'a, N, T>, XwmqRx<'a, N, T>)

初始化消息队列对象

消息队列对象必须调用此方法一次，方可正常使用。 初始化后会返回消息队列的发送者 XwmqTx 以及接收者 XwmqRx 。

示例
use xwrust::xwmd::xwmq::*;

static MQ: Xwmq<16, String> = Xwmq::new();

pub fn xwrust_example_xwmq() {
    let (tx, rx) = MQ.init();
}
