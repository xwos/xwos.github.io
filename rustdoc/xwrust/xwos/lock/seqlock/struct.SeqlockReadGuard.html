pub fn read_retry(&self) -> bool

关闭共享读临界区，并检查是否需要重试

  • 当顺序锁的顺序值没有发生变化，返回 [false] ;
  • 当顺序锁的顺序值发生了变化，返回 [true] 。
示例
loop {
    match sqlk.read_begin() {
        Ok(guard) => {
            let val = *guard;
            if !guard.read_retry() {
                break;
            }
        },
        Err(e) => {
            break;
        },
    }
}
