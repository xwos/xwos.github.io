pub fn xfer( &self, txd: &[u8], rxd: &mut [u8], size: &mut XwSz, to: XwTm ) -> XwEr

传输SPI消息

参数说明
  • txd: 发送数据的缓冲区
  • rxd: 接收数据的缓冲区
  • size: 传输的大小
    • (I) 作为输入时，表示缓冲区大小（单位：字节）
    • (O) 作为输出时，返回实际传输的数据大小
  • to: 期望唤醒的时间点
错误码
  • -EBUSY 总线繁忙
  • -EIO 传输错误
  • -ETIMEDOUT 传输超时
