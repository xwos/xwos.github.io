增加玄武模块

  少于1分钟  

如何增加玄武模块

  • 增加模块文件夹
    • 中间件模块：位于 xwmd/ 目录中；
    • 设备驱动模块：位于 xwcd/ 目录中；
    • 电路板模块：位于 $(XuanWuOS_BRD_DIR)/bm/ 目录中，其中 $(XuanWuOS_BRD_DIR) 是电路板目录，也就是执行 make 的目录；
    • 第三方软件模块：位于 xwem/ 目录中；
    • 应用模块：位于 xwam/ 目录中；
    • OEM模块：位于OEM文件夹内；
  • 文件夹名称需要满足模块路径命名规则
  • 在配置文件中增加编译开关并配置为1，各种模块的配置文件路径如下：
    • 中间件模块： $(XuanWuOS_BRD_DIR)/cfg/xwmd.h ；
    • 设备驱动模块： $(XuanWuOS_BRD_DIR)/cfg/xwcd.h ；
    • 电路板模块： $(XuanWuOS_BRD_DIR)/cfg/board.h ；
    • 第三方软件模块： $(XuanWuOS_BRD_DIR)/cfg/xwem.h ；
    • 应用模块： $(XuanWuOS_BRD_DIR)/cfg/xwam.h ；
    • OEM模块： $(XuanWuOS_BRD_DIR)/cfg/oem.h ；
  • 编译开关 需满足模块编译开关的命名规则
  • 编写编译规则 xwmo.mk ，参考xwmo.mk模板
  • 增加源码；
  • 可以借助辅助功能进行验证（以电路板WeActMiniStm32H750中编译 xwem/vm/lua 为例）：
source xwbd/WeActMiniStm32H750/env.sh
xwcroot # 切换到根路径
cd xwem/vm/lua
xwmc # 获取编译开关（可用于检查配置文件中编译开关是否书写正确）
xwmm # 编译

如何定义OEM文件夹

  • 配置文件： $(XuanWuOS_BRD_DIR)/cfg/XuanWuOS.h
  • 宏： XuanWuOS_CFG_OEMPATH
  • 路径说明
    • 此路径通常配置为相对路径，相对于 电路板文件夹 $(XuanWuOS_BRD_DIR) ；
    • 此路径也可以是绝对路径；
    • 此路径可以是任意路径，并不一定要在XWOS内部。
  • 举例
    • 工程：S32K144Evb
    • 下载： git clone --recursive https://gitee.com/xwos/S32K144Evb.git
    • $(XuanWuOS_BRD_DIR) 是 brd ， XuanWuOS_CFG_OEMPATH 被定义为 .. ，OEM文件夹是 brd/.. ，也即是工程的根目录。XWOS在此工程中作为子目录存在，OEM目录还可作为XWOS的父目录。
